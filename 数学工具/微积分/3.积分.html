                        # 积分

                        积分是求导的逆运算。

                        # 定积分

                        ∫abf(x)dx \int_a^b{f(x)dx} ∫ab​f(x)dx

                        # 不定积分

                        ∫f(x)dx \int f(x)dx ∫f(x)dx

                        # 微积分基本定理

                        ∫abf(x)dx=F(a)−F(b) \int_a^b{f(x)dx} = F(a) - F(b) ∫ab​f(x)dx=F(a)−F(b)

                        # 积分与面积的关系

                        定积分的作用就是求函数的面积。

                        不定积分的作用是求积分函数，比如已知速度函数求路程函数。
