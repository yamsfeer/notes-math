                                          # 命题逻辑

                                          # 命题 Proposition

                                          可以判断真假的陈述句。

                                          逻辑的目的不在于判断命题是否为真，而是在于用逻辑推理判断。

                                          命题分为

                                          • 原子命题
                                          • 复合命题

                                          # 逻辑联接词

                                          联结原子命题形成符合复合命题。

                                          ConnectiveSymbol
                                          and\and\and\and
                                          or\or\or\or
                                          not¬\lnot¬
                                          if-then→\to→
                                          if-and-only-if↔\leftrightarrow↔

                                          # 真值表

                                          # AND Conjunction 合取

                                          pppqqqp\andqp \and qp\andq
                                          TTTTTTTTT
                                          TTTFFFFFF
                                          FFFTTTFFF
                                          FFFFFFFFF

                                          p 和 q 都为真时，p\andqp \and qp\andq 为真。

                                          # OR Disjunction 析取

                                          包含或 inclusive-or ( 逻辑或 or )：允许 p q 同时成立的情况，这是或的默认情况。

                                          pppqqqp\orqp \or qp\orq
                                          TTTTTTTTT
                                          TTTFFFTTT
                                          FFFTTTTTT
                                          FFFFFFFFF

                                          # XOR

                                          排斥或 exclusive-or ( 异或 xor )：不包含 p q 同时成立的情况，相当于 p q 不同时为真。

                                          pppqqqp⊕qp \oplus qp⊕q
                                          TTTTTTTTT
                                          TTTFFFFFF
                                          FFFTTTFFF
                                          FFFFFFFFF

                                          # not

                                          ppp¬p\lnot p¬p
                                          TTTFFF
                                          FFFTTT

                                          # if-then

                                          pppqqqp→qp \to qp→q
                                          TTTTTTTTT
                                          TTTFFFFFF
                                          FFFTTTTTT
                                          FFFFFFTTT

                                          注意这是逻辑学中的 if-then，不是计算机中的 if-then，它是一个控制结构，且它和自然语言的如果-就也有区别。

                                          # if-and-only-if

                                          等价联结词。

                                          pppqqqp↔qp \leftrightarrow qp↔q
                                          TTTTTTTTT
                                          TTTFFFFFF
                                          FFFTTTFFF
                                          FFFFFFTTT

                                          意思是双向箭头前后的 p,qp,qp,q 真值一样时，p↔qp \leftrightarrow qp↔q 真值为 TTT。

                                          意义是“充分必要条件”。

                                          # 复合命题

                                          (p\and¬q)\orq (p \and \lnot q ) \or q (p\and¬q)\orq

                                          ¬\lnot¬ 的优先级最高

                                          复合命题的真值表。

                                          逻辑表达式 与 程序的逻辑表达式

                                          逻辑表达式 + 条件 = 命题

                                          程序逻辑表达式 + 运行时 = 真值

                                          按位与运算与逻辑与运算的区别。

                                          永真式

                                          永假式

                                          # 逻辑等值

                                          逻辑等值就是两个表达式的真值表相同。用 ≡\equiv≡ 表示。

                                          if-and-only-if 用 ↔\leftrightarrow↔ 表示，逻辑等值和 if-and-only-if 有一点区别。区别在于 ↔\leftrightarrow↔ 是一个逻辑表达式，而逻辑等式是两个逻辑表达式。

                                          # 判断一个表达式

                                          两种方式：

                                          • 真值表列出所有值
                                            • 优点：一定能得到所有真值
                                            • 缺点：条件多时，真值表很庞大
                                          • 用逻辑定律化简
                                            • 优点：不用列出所有真值
                                            • 缺点：不一定能推出结果
